Урок 15: Git и GitHub

Что такое Git и GitHub?

Git - это система контроля версий, которая позволяет отслеживать изменения в файлах и координировать работу над проектами. GitHub - это веб-сервис для хостинга Git-репозиториев и совместной работы над проектами.

Основные команды Git

  • git init - инициализация репозитория
  • git status - проверка состояния репозитория
  • git add - добавление файлов в зону подготовленных файлов
  • git commit -m "comment" - создание коммита с комментарием
  • git log - просмотр истории коммитов
  • git diff - просмотр изменений в файлах
  • git checkout - возврат к предыдущему коммиту
  • git branch - создание и управление ветками
  • git merge - слияние веток
  • git remote add origin <url> - добавление удаленного репозитория
  • git push -u origin master - отправка изменений в удаленный репозиторий
  • git pull - получение изменений из удаленного репозитория
